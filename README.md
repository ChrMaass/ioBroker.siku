# ioBroker.siku

Dieser Adapter wird für SIKU RV V2 Lüftungsgeräte entwickelt.

Der aktuelle Entwicklungsfokus liegt auf:

- Multi-Device-Unterstützung in einer Instanz
- UDP-Protokoll gemäß Herstellerdokumentation
- Broadcast-Discovery im lokalen Netzwerk
- zeitgesteuerter RTC-Prüfung und Zeitsynchronisation

Die eigentliche Adapter-Struktur wird im nächsten Schritt mit dem offiziellen ioBroker-Scaffold erzeugt.
